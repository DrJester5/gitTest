Tags and usage

TAGSUSAGEREQURIES CLOSE
<span>Defines a section in a document, can be within a paragraph and does not default CSSYES
<aside>Defines content aside from the page contentYES
<h1>Defines a header/title within a document - Uses h1-h6 with h1 the largest and h6 the smallest.YES
<link>Defines the relationship between a document and an external resource (most used to link to style sheets). This tag is placed within the 'HEAD' tagNO
<caption>Defines a table caption, used like a h1 or header, but specifically used for tablesYES
<video>Defines embedded video content, requires a 'src' file to link toYES
<ol> and <ul>Used to create lists, 'ol' is an ordered list, where 'ul' defines an un-ordered listYES

CSS selectors and specificity

CSS SELECTORUSAGEOUTCOME
.class name {}This is a 'CLASS SELECTOR' and links to a class name within the HTML document with the use of '.' and the class name used in the HTML documentWill select only where this class name is used and add style to the element/s - Classes follow ID's for specificity
div {}Selects the 'div' elements of the HTML documetWill style all 'div' elements within the document
.altrow td {}This is a 'Descendant Combinator' and selects all the 'td' elements within the class of 'altrow'Will add style to the table data with a class of altrow
h1, h2, h3, p {}This is 'CHAINING' and will select all of the 'h1', 'h2', 'h3' and 'p' tags from the HTML documentWill style all of the selected tags to the same CSS rules
#id name {}This is an 'ID SELECTOR' and links to a ID name within the HTML document using the'#'This will select only where this ID name is used, ID's should only be used if 1 change is specific - ID's are the most specific
p: hover {}This selector is known as a 'PSUDEO CLASS', its like a child selctor of the main selector usedWill do what is chosen when the mouse hovers over a paragraph element
img[src*='winter'] {}This one is an'ATTRIBUTE' selector, the square brackets define the attribute that is selected within the parent selector.Current selector will add style to the images with the attribute 'src' that contains the string 'winter'
* {}UNIVERSAL SELECTOR, shown by a '*'The 'universal selector' will select ALL elements of ANY type targeted within the braces
